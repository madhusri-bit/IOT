⭐ 1. Introduction

NodeMCU is a Wi-Fi enabled microcontroller board used widely in IoT applications. Unlike Arduino, NodeMCU has built-in Wi-Fi, which makes it suitable for cloud connectivity and smart home automation. This experiment teaches the basics of NodeMCU ESP8266, its features, and how to configure it.

⭐ 4. About ESP8266 WiFi SoC

(From page 40–41 

23ITC01-Iot Laboratory Manual F…

)

ESP8266 is a low-cost Wi-Fi chip with:

80/160 MHz processor

Built-in TCP/IP stack

Support for Wi-Fi b/g/n

10-bit ADC

PWM support

3.3V logic pins

NodeMCU board includes:

Micro USB port

Flash button

Reset button

Antenna

GPIO pins

⭐ 5. Pin Configuration Overview

(Page 43 

23ITC01-Iot Laboratory Manual F…

)

Digital pins: D0 to D8

Analog pin: A0

3.3V & GND pins

Special pins for UART, SPI, and I2C

Important note: ESP8266 pins are NOT 5V tolerant.

⭐ 6. How to Use NodeMCU in Arduino IDE

(From pages 44–46 

23ITC01-Iot Laboratory Manual F…

)

Steps:

Open Arduino IDE → Preferences

Add board package URL

Open Boards Manager → Search esp8266 → Install

Select NodeMCU 1.0 (ESP-12E)

Connect NodeMCU to PC

Select COM port

Upload a basic program
